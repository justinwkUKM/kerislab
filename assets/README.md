# KerisLab Asset Pack

This folder contains the source visual assets for KerisLab. The set is vector-first, Apple-inspired, and designed for a professional security platform: light surfaces, black typography, cool gray structure, blue/cyan/indigo accents, subtle translucency, and clean geometry.

## Structure

- `brand/`: primary logo mark, wordmark, and app icon.
- `backgrounds/`: large-format SVG backgrounds for login, dashboard, and Mission Control surfaces.
- `icons/`: product feature icons built on a consistent 64x64 grid.
- `illustrations/`: larger product illustrations for docs, onboarding, and empty states.
- `manifest.md`: asset catalog, intended use, and design tokens.

## Design Rules

- Use these assets on light-first UI surfaces by default.
- Do not recolor severity icons without checking contrast.
- Keep the mark clear of busy backgrounds.
- Preserve the `viewBox` when embedding SVGs.
- Prefer inline SVG for app UI when color tokens need to adapt to theme.
